Help Center

Version History

v1.2.7 [February 9, 2025]

Bug Fixes

  • Bug Fix: If the dropped folders contained special characters like [], the files inside these folders could not be processed. This issue has been fixed.

v1.2.6 [January 24, 2024]

New Features and Bug Fixes

  • New Feature: Added a Help Menu under the Tools menu. Information about program usage and version history can be accessed from this menu.
  • Bug Fix: Prevented running multiple instances of the program simultaneously.

v1.2.5 [January 15, 2025]

Bug Fixes and Improvements

  • Bug Fix: The WinRAR extraction command was modified to handle CBR/CBZ files containing multiple folders with similarly named files, ensuring smooth optimization and PDF conversions.
  • Bug Fix: The correctness of images generated during PDF-to-CBR conversion is now verified.
  • Bug Fix: Third-party helper programs running in the background are now properly terminated if the program is closed during processing.
  • Improvement: Python libraries have been updated.
  • Improvement: A "Processed by CBX Tools" note has been added to WinRAR archives to easily identify optimized archives.
  • Improvement: The parallel file processing feature, removed in version 1.2.1, has been reintroduced.

v1.2.1 [December 10, 2024]

Bug Fixes

  • Bug Fix: During PDF-to-CBR conversion, sequential processing is now used instead of parallel processing for PDFs containing a single image per page to prevent rare errors during JPEG export.
  • Bug Fix: The solid archive status was corrected for Turkish. When the program is used in Turkish, this value was displayed as Yes/No.

v1.2 [October 31, 2024]

Bug Fixes

  • Bug Fix: The program has been modified to work on high-resolution displays (e.g., 4K screens) with Windows scaling settings over 100%.
  • Bug Fix: Existing XML files in the source CBR/CBZ files are now included in the final CBR output during optimization process.

v1.1.6 [October 29, 2024]

Bug Fixes

  • Bug Fix: If the processed file name started with a hyphen (-), WinRAR mistook it for a parameter and returned an error.
  • Bug Fix: If the user activated the solid archive option in the default WinRAR configuration file, WinRAR used this setting regardless of the program's configuration.
  • Bug Fix: Files created on macOS, containing hidden __MACOSX folders for metadata, caused errors during processing.
  • Bug Fix: When using jpegMini Pro for optimization, the information box displayed the "EXTRACT" message from the previous step.

v1.1.3 [June 3, 2024]

Improvements

  • Improvement: All third-party programs except jpegoptim.exe have been replaced with Python libraries.
  • Improvement: Conversion of PDFs with text or multiple images per page to CBR is now 50% faster.
  • Improvement: The license agreement in the setup program was updated to reflect the MIT license.

v1.1.1 [June 1, 2024]

New Features, Bug Fixes, and Improvements

  • New Feature: Total processing time for each operation is now displayed in hours, minutes, and seconds after completion.
  • Bug Fix: Manual and Disabled options in JPEG Optimization were unavailable in Turkish mode.
  • Bug Fix: Reprocessing files from OUTPUT folder is now prevented to avoid errors.
  • Improvement: pdfinfo.exe has been replaced with an equally fast Python library.
  • Improvement: The About Program text has been simplified.
  • Improvement: The program is now licensed under the MIT license.
  • Improvement: The progress bar in the lower-right corner has been shortened and swapped with the adjacent Info Label.
  • Improvement: Non-critical and repetitive messages are now displayed on the Info Label instead of the main output screen.

v1.0.8 [May 1, 2024]

Bug Fixes and Improvements

  • Bug Fix: Code added in version 1.0.7 caused errors during optimization with JPEGMini and has been corrected.
  • Improvement: pdftotext has been replaced with a faster Python library.

v1.0.7 [May 1, 2024]

New Features and Improvements

  • New Feature: Total processing time for each operation is displayed in seconds after completion.
  • Improvement: ImageMagick and QPDF third-party tools have been replaced with Python libraries.
  • Improvement: CBR/CBZ to PDF conversion is now 65% faster compared to the previous version.

v1.0.6 [April 8, 2024]

Improvements

  • Improvement: The two-step alternative method for PDF-to-CBR conversion has been removed, allowing all conversions to be done in a single step.
  • Improvement: A Turkish license text has been added to the setup file.
  • Improvement: The settings.ini file is now preserved during program updates.
  • Improvement: Program version information is now embedded in the exe file for viewing in Windows Explorer.
  • Improvement: Version information has been added to the About menu.

v1.0.5 [April 7, 2024]

Bug Fixes

  • Bug Fix: Typographical errors in some Turkish messages have been corrected.

v1.0.4 [April 6, 2024]

Improvements

  • Improvement: The alternative method for PDF-to-CBR conversion has been accelerated by 50%.

v1.0.3 [April 6, 2024]

Bug Fixes and Improvements

  • Improvement: The installer now automatically removes the previous version before installing the new one.
